# AuthKit

Auth starter: JWT/Refresh, RBAC, Swagger

## Links
- Live (Vercel): https://dev-kaiki-authkit.vercel.app
- Swagger (Render): https://dev-kaiki-authkit-api.onrender.com/docs
- Health: https://dev-kaiki-authkit-api.onrender.com/health

## One-click deploy

API (Render Blueprint):
https://render.com/deploy?repo=https://github.com/dev-kaiki/authkit

WEB (Vercel):
https://vercel.com/new/clone?repository-url=https://github.com/dev-kaiki/authkit&project-name=dev-kaiki-authkit&repository-name=authkit&root-directory=apps/web

Vercel ENV:
NEXT_PUBLIC_API_URL=https://dev-kaiki-authkit-api.onrender.com

## Local (Windows)
Run:
powershell -ExecutionPolicy Bypass -File .\dev.ps1
